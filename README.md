# Front-end
Welcome to ClearVote's front-end repo. Our tech stack includes `React.js/Next.js`, `Vercel`, `TailwindCSS`, `Typescript`, and `ShadcnUI`.

To get started, clone this repo and type this into your console:
`cd next-app`
`npx next dev`
